# Operator Guide

## First Deployment
1. Register operator, sites/contracts, guard licenses, personnel and robots.
2. Import existing dispatch and incident-reporting history.
3. Run read-only license-scope and patrol-robot mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run incident report and audit export.

## Minimum Production Controls
- guard license-scope validation before any dispatch
- governor gate on every robot/guard action before dispatch
- human sign-off for :high/:safety-critical actions (use of force, private
  residence entry, detaining a person)
- evidence-backed incident reports
- audit export for every dispatch, sign-off and report
- backup manual dispatch process

## Certification
Certified operators must prove robot-safety integrity, guard-license
discipline, evidence-backed incident reporting and human review for
force-affecting actions.
